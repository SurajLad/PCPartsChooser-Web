# PCPartsChooser-Web

Its was My SEM 6 Lab Manual Mini Project For Development of Dynamic Website Having Subject of PCPartsChooser To Build PC as per your Required Configuration

Technology Used:- 
> HTML,Javascript,MySQL,php,JQuery
